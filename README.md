# Steps to run the project

Clone the project from github using the command, git clone https://github.com/vignesh387/financial-services.git.

Install dependency using the command, npm install.

Run the app on local server using the command, npm run dev.

# Setup open ai api

Create an open ai account in this link https://auth.openai.com/log-in and log in.

Check credit balance in this link https://platform.openai.com/usage.

If no credit balance, api request cannot be successfull, add credit balance in this link https://platform.openai.com/settings/organization/billing/overview

Go to api keys page using the link https://platform.openai.com/api-keys and click create new secret key button
to create new api key.

Copy the api key and assign it to the VITE_API_KEY variable in env file.
